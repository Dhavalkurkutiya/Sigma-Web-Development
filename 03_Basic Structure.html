Basic Structure

Every HTML document consists of the following basic structure:

  • DOCTYPE declaration
  • HTML element
  • Head element
  • Body element

DOCTYPE Declaration

The DOCTYPE declaration is the first line of an HTML document and tells the browser which version of HTML the document is written in. It is not an HTML tag, but rather an instruction to the browser. The DOCTYPE declaration for HTML5 is as follows: 

<!DOCTYPE html>

HTML Element

The HTML element is the root element of an HTML document and contains all other elements. It is denoted by the opening and closing <html> tags. 

<html>
    ...
</html>

Head Element

The head element contains meta-information about the document, such as its title, links to stylesheets, and scripts. It is denoted by the opening and closing <head> tags. 

<head>
        ...
    </head>

Body Element

The body element contains the content of the document, such as text, images, links, and other elements. It is denoted by the opening and closing <body> tags. 

<body>
            ...
        </body>
