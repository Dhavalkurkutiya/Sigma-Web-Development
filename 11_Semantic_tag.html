Semantic Tags

Semantic elements are those which clearly describe their meaning in a human- and machine-readable way. They are used to define the structure of the web page. Some of the semantic elements are:

Header

The header element represents a container for introductory content or a set of navigational links.

Nav

The nav element represents a section of a page that links to other pages or to parts within the page: a section with navigation links.

Section

The section element represents a generic section of a document or application. A section, in this context, is a thematic grouping of content, typically with a heading.

Article

The article element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable.

Aside

The aside element represents a portion of a document whose content is only indirectly related to the document's main content.

Footer

The footer element represents a footer for its nearest sectioning content or sectioning root element. A footer typically contains information about the author of the section, copyright data, or links to related documents.
