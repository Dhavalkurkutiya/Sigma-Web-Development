Css Selectors

There are several types of selectors in CSS that are used to select the HTML elements and apply styles to them. The selectors are:

  • Element Selector
  • Id Selector
  • Class Selector
  • Universal Selector
  • Group Selector
  • Descendant Selector
  • Child Selector
  • Adjacent Sibling Selector
  • General Sibling Selector

Element Selector

The element selector selects the HTML element and applies the styles to all the elements of the same type. The syntax of the element selector is: 

        
            element_name{
                property: value;
            }

Id Selector

The id selector selects the HTML element with a specific id attribute and applies the styles to that element. The syntax of the id selector is: 

        
            #id_name{
                property: value;
            }

Class Selector

The class selector selects the HTML element with a specific class attribute and applies the styles to that element. The syntax of the class selector is: 

        
            .class_name{
                property: value;
            }

Universal Selector

The universal selector selects all the HTML elements on the web page and applies the styles to all the elements. The syntax of the universal selector is: 

        
            *{
                property: value;
            }

Group Selector

The group selector selects multiple HTML elements and applies the styles to all the elements. The syntax of the group selector is: 

        
            element1, element2, element3{
                property: value;
            }

Descendant Selector

The descendant selector selects the HTML element that is a descendant of another element and applies the styles to that element. The syntax of the descendant selector is: 

        
            element1 element2{
                property: value;
            }

Child Selector

The child selector selects the HTML element that is a child of another element and applies the styles to that element. The syntax of the child selector is: 

        
            element1 > element2{
                property: value;
            }

Adjacent Sibling Selector

The adjacent sibling selector selects the HTML element that is the adjacent sibling of another element and applies the styles to that element. The syntax of the adjacent sibling selector is: 

        
            element1 + element2{
                property: value;
            }

General Sibling Selector

The general sibling selector selects the HTML element that is the sibling of another element and applies the styles to that element. The syntax of the general sibling selector is: 

        
            element1 ~ element2{
                property: value;
            }
